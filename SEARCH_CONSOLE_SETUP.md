# Google Search Console setup

1. Add this URL-prefix property:
   https://hyuksookwon.github.io/

2. Choose HTML file upload.

3. Download the unique file named something like:
   google1234567890abcdef.html

4. Put that file in the repository root, beside index.html.

5. Commit and push.

6. Check that it opens at:
   https://hyuksookwon.github.io/google1234567890abcdef.html

7. Return to Search Console and click Verify.

Do not remove the Google verification file after verification.

After verification, submit:
sitemap.xml
